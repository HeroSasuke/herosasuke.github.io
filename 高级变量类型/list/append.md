# append

|语法|用法|描述|
|:|:|:|
|append|list.append(object)|通过append可以向列表添加元素|

实例:

```python
In [56]: name_list = ["zhangsan", "lisi"]

In [57]: name_list.append("wangwu")

In [58]: name_list
Out[58]: ['zhangsan', 'lisi', 'wangwu']

In [59]: name_list.append("python")

In [60]: name_list
Out[60]: ['zhangsan', 'lisi', 'wangwu', 'python']
```
