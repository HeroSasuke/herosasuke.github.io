# count

   </tr>
    <tr>
       <td>count</td>
       <td>count(object)</td>
       <td>统计元素在列表中出现的次数</td>
    </tr>

</table>

实例:
```python
namelist = ["zhangsan", "lisi", "zhangsan", "wangwu", "zhaoliu", "zhangsan"]

index = namelist.count("zhangsan")

print(index)
```
运行结果:
```
3
```
