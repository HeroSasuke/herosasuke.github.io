# insert

|语法|用法|描述|
|:|:|:|
|insert|list.insert(index, object)|insert(index, object) 在指定位置index前插入元素object|

实例:

```python
In [67]: nums = [0, 1, 2, 3]

In [68]: nums.insert(1, 4)

In [69]: nums
Out[69]: [0, 4, 1, 2, 3]

In [70]: nums.insert(0, 5)

In [71]: nums
Out[71]: [5, 0, 4, 1, 2, 3]
```
