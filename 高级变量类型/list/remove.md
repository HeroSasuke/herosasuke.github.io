# remove

<table>
    <tr>
      <td><b>语法</b></td>
      <td><b>用法</b></td>
      <td><b>描述</b></td>
   </tr>
  <tr>
    <td>remove</td>
    <td>list.remove(object)</td>
    <td>会删除列表中首次出现的指定的数据</td>
 </tr>
</table>

实例:
```python
namelist = ["zhangsan", "lisi", "wangwu", "zhaoliu"]

namelist.remove("zhangsan")

print(namelist)
```

运行结果:
```
['lisi', 'wangwu', 'zhaoliu']
```
