实例:

  ```python
  namelist = [4, 2, 1, 5, 10, 9, 8, 6]

  namelist.reverse()

  print(namelist)
  ```
运行效果:

  ```python
  [6, 8, 9, 10, 5, 1, 2, 4]
  ```
