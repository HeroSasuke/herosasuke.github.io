# 修改元素

|语法|用法|描述|
|:|:|:|
|根据索引修改|列表[index] = object|修改指定索引的数据|

实例:
```python
In [78]: name_list = ["zhangsan", "lisi", "wangwu", "zhaoliu"]

In [79]: name_list[0] = "张三"

In [80]: name_list
Out[80]: ['张三', 'lisi', 'wangwu', 'zhaoliu']

In [81]: name_list[3] = "赵六"

In [82]: name_list
Out[82]: ['张三', 'lisi', 'wangwu', '赵六']
```
