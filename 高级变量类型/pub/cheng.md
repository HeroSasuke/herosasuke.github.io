# *

|运算符|Python 表达式|结果|描述|支持的数据类型|
| :--| :--| :--|:--|:--|
|*	|['Hi!'] * 4	|['Hi!', 'Hi!', 'Hi!', 'Hi!']	|复制	|字符串、列表、元组|

实例:

```python
In [10]: "hello " * 3
Out[10]: 'hello hello hello '

In [11]: [1, 2] * 3
Out[11]: [1, 2, 1, 2, 1, 2]

In [12]: (1, 2) * 3
Out[12]: (1, 2, 1, 2, 1, 2)
```
