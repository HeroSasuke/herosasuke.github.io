# +

|运算符|Python 表达式|结果|描述|支持的数据类型|
| :--| :--| :--|:--|:--|
|+	|[1, 2] + [3, 4]|	[1, 2, 3, 4]	|合并	|字符串、列表、元组|

实例:
```python
In [84]: "hello" + "python"
Out[84]: 'hellopython'

In [85]: [1, 2] + [3, 4]
Out[85]: [1, 2, 3, 4]

In [86]: ("a", "b") + ("c", "d")
Out[86]: ('a', 'b', 'c', 'd')
```
