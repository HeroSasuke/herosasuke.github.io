# max

|序号 | 方法 |描述|
| :--| :--|:--|
|3	|max(item)	|返回容器中元素最大值|


### 语法

```
max( x, y, z, .... )
```

### 参数
- x -- 数值表达式。

- y -- 数值表达式。

- z -- 数值表达式。

### 返回值
```
返回给定参数的最大值。
```

### 实例:
```Python
In [2]: max("hello itcast")
Out[2]: 't'

In [3]: max([1,4,522,3,4])
Out[3]: 522

In [4]:  max({"a":1, "b":2})
Out[4]: 'b'

In [5]:  max({"a":10, "b":2})
Out[5]: 'b'

In [6]: max({"c":10, "b":2})
Out[6]: 'c'
```
