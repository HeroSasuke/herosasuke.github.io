# isalpha

|语法|用法|描述|
|:|:|:|
|isalpha|mystr.isalpha()  |如果 mystr 所有字符都是字母 则返回 True,否则返回 False|

实例:

```python
In [13]: mystr = "abc"

In [14]: mystr.isalpha()
Out[14]: True

In [15]: mystr = "123"

In [16]: mystr.isalpha()
Out[16]: False

In [17]: mystr = "abc123"

In [18]: mystr.isalpha()
Out[18]: False
```
