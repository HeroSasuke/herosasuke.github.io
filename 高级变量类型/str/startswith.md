# startswith

|语法|用法|描述|
|:|:|:|
|startswith|mystr.startswith(hello)|检查字符串是否是以 hello 开头, 是则返回 True，否则返回 False|

实例:

```Python
In [33]: mystr = 'hello world itcast and itcastcpp'

In [34]: mystr.startswith("hello")
Out[34]: True

In [35]: mystr.startswith("Hello")
Out[35]: False
```
